RAG Legal Local

Buscador inteligente sobre tus documentos legales — Word + PDF · Qdrant + SQLite · DeepSeek + Groq

⚡ Instalación en 3 comandos

git clone https://github.com/mmansillaf/rag-legal-local.git
cd rag-legal-local
pip install -r requirements.txt

Después: python ingestion_pipeline.py --input "C:\tus\docs"

Interfaz web: streamlit run app.py → doble-click en RAG-Legal.bat

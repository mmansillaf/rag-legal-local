Guía de Instalación
RAG Legal Local en Windows

Buscador inteligente sobre tus documentos legales (Word + PDF)

¿Necesito instalar Hermes Agent?

NO. El sistema funciona standalone con query_pipeline.py. Hermes Agent es opcional y aporta web fallback, memoria y clasificacion de preguntas. Empeza sin Hermes.

Requisitos

RequisitoDetalle
Windows10 u 11, 64-bit
RAM8 GB mínimo, 16 GB recomendado
Disco4 GB libres para índices + espacio de tus documentos
Python3.10 o superior
API KeysDeepSeek (pay-as-you-go) o Groq (free tier)

Pasos de Instalación

  1. Instalar Python

    Descargar de python.org, ejecutar el instalador, y marcar "Add Python to PATH". Verificar:

    python --version

  2. Descargar el proyecto

    cd C:\Users\TU_USUARIO\
git clone https://github.com/mmansillaf/rag-legal-local.git
cd rag-legal-local

    Sin Git: git-scm.com/download/win

  3. Instalar dependencias

    pip install -r requirements.txt

    La primera vez descarga el modelo bge-m3 (~2 GB). Solo ocurre una vez.

  4. Configurar API Keys

    Conseguí tus keys en:

    DeepSeek: platform.deepseek.com → recargar $2 (dura meses)

    Groq: console.groq.com → free tier

    $env:DEEPSEEK_API_KEY = "sk-tu-key"
$env:GROQ_API_KEY = "gsk_tu-key"

    Para que sea permanente: Win+R → sysdm.cpl → Avanzadas → Variables de entorno → Nueva

  5. Indexar tus documentos

    python ingestion_pipeline.py --input "C:\Users\TU_USUARIO\Documentos\Legales"
    100 docs2-5 minutos
    1,000 docs15-30 minutos
    10,000 docs1-2 horas

    Solo se hace UNA vez. Podés seguir usando la PC mientras tanto.

  6. Hacer tu primera consulta

    python query_pipeline.py "cual es el plazo de apelacion?"

Uso Diario

Agregar documentos nuevos

python ingestion_pipeline.py --input "C:\...\Legales" --incremental

Modo chat interactivo

python query_pipeline.py --interactive

Solución de Problemas

"No module named 'markitdown'"

pip install markitdown

"No module named 'utils'"

Ejecuta los scripts desde la carpeta rag-legal-local.

Indexación muy lenta

Normal para miles de docs. Cerrá otras apps, usá --batch-size 50.
